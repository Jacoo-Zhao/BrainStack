Code and Data Availability
--------------------------

This submission package contains the implementation of our proposed framework, including:

- **Core Code**:
  - `main.py`, `model.py`, `loss.py`, `utils.py` – model architecture and loss functions
  - `trainer/` – training pipeline (construct_model.py, train.py, validate.py)
  - `options.py` – argument parsing and configuration
  - `requirements.txt` – environment dependencies

- **Documentation**:
  - `README.md` – usage instructions
  - `assets/` – figures and supplementary materials

- **Data**:
  The full SS-EEG dataset is currently under preparation and anonymization, and therefore is not included in this submission package.
  To facilitate reproducibility during review, we will upload small preview samples (synthetic inputs and preprocessing examples) to the GitHub repository.

- **Future Release**:
  Upon acceptance, we will publicly release:
  1. SS-EEG dataset
  2. Preprocessing scripts
  3. Detailed training and evaluation instructions
